# Uebung 25: Gesamtcheck Best Practices

Ziel: Dockerfile, Compose und Kubernetes Deployment gemeinsam analysieren.

## Dateien

- `Dockerfile`: Beispiel mit Optimierungspotenzial
- `compose.yaml`: Beispiel mit Konfigurationsproblemen
- `deployment.yaml`: Beispiel ohne wichtige Stabilitaetsmerkmale

## Aufgaben

1. Finde in jeder Datei mindestens 2 Verbesserungsmoeglichkeiten.
2. Ordne jede Stelle einer Kategorie zu: Build, Sicherheit, Konfiguration oder Stabilitaet.
3. Diskutiere, welche Aenderung im Alltag den groessten Nutzen bringen wuerde.
