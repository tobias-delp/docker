# Uebung 21: Dockerfile Best Practices

Ziel: Ein einfaches Dockerfile mit einer verbesserten Variante vergleichen.

## Dateien

- `Dockerfile.bad`: unguenstige Ausgangsversion
- `Dockerfile.good`: optimierte Variante
- `package.json`: minimales Node-Beispiel
- `app.js`: kleine Anwendung

## Aufgaben

1. Vergleiche beide Dockerfiles.
2. Benenne Unterschiede bei Basisimage, Layering und Multi-Stage Build.
3. Erklaere, warum die optimierte Variante im Betrieb besser ist.
